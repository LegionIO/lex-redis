# lex-redis

Redis integration for [LegionIO](https://github.com/LegionIO/LegionIO). Run key/value operations and server management commands against Redis servers.

## Installation

```bash
gem install lex-redis
```

## Functions

- **Item** - Key/value operations (get, set, delete, etc.)
- **Server** - Server management and info

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
- Redis server
